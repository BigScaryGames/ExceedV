## Damage Formula
`Damage = Weapon Dice + Might + Bonuses`
**MS5 PLaceholder for crits.**
`Criticals = Damage * 2`
## Damage Dice by Skill Level

| Skill Level | Damage Dice |
|-------------|-------------|
| 1-2 | 1 die |
| 3-4 | 2 dice |
| 5+ | 3 dice |

## Weapon Dice by Weight

| Weight | Typical Dice |
|--------|--------------|
| Light | d4/d6 |
| Normal | d6/d8 |
| Heavy | d8/d12 |

## Offensive Attributes

All melee weapons can use **Agility** as the main stat for offensive maneuvers.

- **Heavy:** Allows using Might (generally heavy, blunt weapons)
- **Finesse:** Allows using Dexterity (generally piercing/slashing weapons)
- **Ranged:** Uses Dexterity or Perception based on weapon type

Certain perks allow you to change the used attribute.

## Damage Application

Damage depletes Stamina first, then Health. See [[3.3 HP And Wounds]] for details.
