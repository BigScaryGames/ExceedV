# Defense Traits

*Determine which defenses can be used against an ability*

| Trait       | Valid Defenses                          | Description                                 |
| ----------- | --------------------------------------- | ------------------------------------------- |
| #Strike     | Deflect, Dodge                          | Melee attacks, touch spells                 |
| #Projectile | Dodge, Deflect (with a shield or perks) | Ranged attacks, thrown weapons              |
| #Burst      | Dodge, Deflect (with perks)             | Explosions, area effects                    |
| #Mind       | Resolve                                 | Mental effects: fear, charm, mind control   |
| #Body       | Endure                                  | Bodily effects: poison, disease, exhaustion |
