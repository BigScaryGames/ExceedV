**Base Action Points:** 5 AP per turn + 1 Reaction

**Bonus Action Points:**
- **Mental AP:** Every 5 total Mental attributes (Perception + Will + Charisma + Wit) = +1 Mental AP
- **Physical AP:** Every 5 total Physical attributes (Might + Endurance + Agility + Dexterity) = +1 Physical AP

**AP Type Restrictions:**
- Mental AP can only be used for mental actions: spells, social interactions, knowledge checks, perception, concentration
- Physical AP can only be used for physical actions: movement, attacks, athletics, manual tasks

**Reactions:**
Using a reaction reduces the number of AP at the start of the next turn by 1. If you have Reactive or other perks allowing multiple reactions, reduce the number of actions by the number of reactions used.
