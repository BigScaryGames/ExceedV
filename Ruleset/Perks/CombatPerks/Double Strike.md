# Double Strike

 **Requirement:** 1H1
 **Attributes:** AG/MG+1H
 **AP Cost:** 0
 **XP Cost:** 5
 **Tags:** #OneHanded

## Short Description
Strike with both weapons, but second misses if first is blocked.

## Effect
Strike with both weapons with -2 AP. If the enemy successfully defends against the first strike, the second misses automatically.
